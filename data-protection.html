Data Protection

PLASSX applies strict data protection principles aligned with international standards.

1. Data Protection Commitment

We ensure:

  • Lawful and transparent processing
  • Data minimization
  • Purpose limitation
  • Security and integrity

2. Technical Measures

  • Encryption protocols
  • Access control systems
  • Secure hosting infrastructure
  • Activity monitoring

3. Organizational Measures

  • Restricted internal access
  • Confidentiality agreements
  • Data handling protocols

4. Third-Party Processing

All third-party providers must comply with:

  • Data protection standards
  • Confidentiality requirements

5. Incident Response

In case of a data breach:

  • Immediate containment measures will be taken
  • Affected users may be notified if required

6. User Data Rights

Users may request:

  • Data access
  • Data correction
  • Data deletion (subject to legal limitations)
